# Portfolio Premium - Full Stack Developer

Portfólio de alto padrão para desenvolvedores full stack, com estética moderna, animações elegantes e visual memorável.

## Tecnologias

- **Next.js 16** - React Framework
- **TypeScript** - Tipagem estática
- **Tailwind CSS v4** - Estilização
- **Framer Motion** - Animações
- **Lucide React** - Ícones

## Design

- Tema dark premium
- Glassmorphism
- Gradientes modernos
- Animações suaves
- Layout responsivo
- Header sticky com blur
- Scroll suave

## Início Rápido

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Personalização

Edite `src/data/config.ts` para customizar:
- Nome e informações pessoais
- Email e WhatsApp
- Links de redes sociais
- Projetos em destaque

## Estrutura

```
src/
├── app/
│   ├── layout.tsx    # Layout principal
│   ├── page.tsx      # Página inicial
│   └── globals.css   # Estilos globais
├── components/       # Componentes React
│   ├── Header.tsx
│   ├── Hero.tsx
│   ├── About.tsx
│   ├── Stack.tsx
│   ├── Projects.tsx
│   ├── Differentiators.tsx
│   ├── Contact.tsx
│   └── Footer.tsx
└── data/
    └── config.ts     # Configuração centralizada
```

## Build

```bash
npm run build
npm start
```
